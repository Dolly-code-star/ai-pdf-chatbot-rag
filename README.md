# AI PDF Chatbot (RAG)

## Project Overview

This project is part of my AI Engineer Portfolio.

It demonstrates the first stages of a Retrieval-Augmented Generation (RAG) pipeline:

- Read PDF documents
- Extract text
- Split text into chunks

## Tech Stack

- PyPDF
- Git
- GitHub
